A header row in a table contains the names of the columns in the table.

We can use `<th>` in place of `<td>` for making a header row.

<codeblock language="html" type="lesson">
<code>
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>Gender</th>
  </tr>
  <tr>
    <td>New</td>
    <td>24</td>
    <td>Male</td>
  </tr>
  <tr>
    <td>New</td>
    <td>31</td>
    <td>Female</td>
  </tr>
</table>
</code>
</codeblock>
